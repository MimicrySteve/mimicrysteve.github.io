MIMICRY is the first software to help artists easily create Infinite Animation (IA) artwork - a novel type of artistic medium which utilizes artificial intelligence as a consistent (background) tool in further expressing an artist’s message.
Infinite Animation, through the use of AI, displays a constantly updating scene: never showing the same information twice. Each piece takes on a life of it’s own. This requires the artist to have a vision not just for the present image but for how it will develop into the future. Click here to learn more about Infinite Animation artwork.
Envisioning Real-time Data
One of the most obvious artistic themes facilitated by this platform is the incoorporation of realtime data - updated in the last day, hour, or minute - producing visualizations of this never seen before information. Government releases, market data, even personal health info from a wearable can be incoorperated into pieces realtime.

This unlocks art which is reactive - art which is alive.
Flexible Interface
MIMICRY provides a streamlined interface. Providing GUI options for common artistic patterns. Without coding a single line, the interface allows the users to make Infinite Animations with just a few clicks.

For advanced users, MIMICRY provides a full text editor for writing more complex systems in python. Code is dynamically imported providing the ability to rapidly test these patterns.
Final export is always into HTML code. Making all art capable of viewing on every device: TV screansavers, smartphones viewing, or on dedicated computers for particularly robust pieces.

Built-in Support For NFTs
MIMICRY is built from the ground up to support NFT generation. The artist has full control over the unique production of NFTs from the second of creation. The software is built to provide full control to the artist. Decide
- How NFTs are owned
- What rights to retain and/or sell
- How the art is viewed
- The production of second generation NFTs
- Royalty and trasfer rates
